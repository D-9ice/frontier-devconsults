# Project Logos

Place your project logos in this directory with the following names:

## Required Logo Files:

1. **frontier-logo.png** - Frontier Development Consultants logo (512x512px) ✓
2. **czan-logo.png** - C-ZAN Guest House & Lounge logo (512x512px) ✓
3. **macsunny-logo.png** - MacSunny Electronics logo (512x512px) ✓
4. **lotto-logo.png** - Lotto Forecaster AI logo (512x512px) ✓

## Logo Guidelines:

- **Format**: PNG with transparent background preferred
- **Size**: Square images work best (1:1 ratio)
- **Resolution**: Minimum 256x256px, recommended 512x512px or higher
- **File Size**: Keep under 500KB for optimal loading

## Current Logo Paths:

- Frontier Development Consultants: `/logos/frontier-logo.png`
- C-ZAN Guest House: `/logos/czan-logo.png`
- MacSunny Electronics: `/logos/macsunny-logo.png`
- Lotto Forecaster AI: `/logos/lotto-logo.png`

Until actual logos are added, the cards will display color-coded icon placeholders.
