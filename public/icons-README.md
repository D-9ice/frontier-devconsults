# Placeholder for PWA icons
# These icons are required for the Progressive Web App functionality

## Required Icons:
- icon-192.png (192x192px) - For mobile home screen
- icon-512.png (512x512px) - For splash screen and larger displays  
- og-image.png (1200x630px) - For social media sharing (Open Graph)
- logo.png (256x256px or higher) - Company logo

## Instructions:
1. Create or export your Frontier DevConsults logo
2. Generate the following sizes:
   - 192x192px (save as icon-192.png)
   - 512x512px (save as icon-512.png)
   - 1200x630px (save as og-image.png) - can include text/branding
   - 256x256px+ (save as logo.png)

3. Place all files in the /public directory

## Tools for generating icons:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
- Photoshop/Figma/Canva for custom design

## Format:
- PNG format with transparent background (for icons)
- Square aspect ratio for icon-192 and icon-512
- 1200:630 aspect ratio for og-image
